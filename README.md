# Health Office Assistant

Background health reminder daemon for office workers. Sends periodic notifications to stretch, hydrate, rest eyes, and maintain good posture.

## Install

```bash
uv sync              # install dependencies
```

## Usage

```bash
uv run hoa run       # start daemon
uv run hoa tui       # TUI dashboard
uv run hoa status    # quick progress
uv run hoa config    # view settings
```
