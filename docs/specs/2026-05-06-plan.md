# Health Office Assistant (HOA) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a Python CLI tool that sends interactive Windows toast notifications for water/sedentary/eye reminders during work hours, with idle detection and an optional TUI dashboard.

**Architecture:** Background daemon with APScheduler managing 3 independent timers, windows-toasts for interactive notifications, ctypes-based idle detection, JSON file for state persistence, and a Textual TUI that reads the same state file.

**Tech Stack:** Python 3.12+, windows-toasts, textual, apscheduler, click (CLI), ctypes (Win32 API)
