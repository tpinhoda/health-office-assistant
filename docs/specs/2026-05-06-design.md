# Health Office Assistant (HOA) — Design Spec

**Date**: 2026-05-06
**Status**: Approved
**Author**: Tiago

---

## Overview

A lightweight Windows desktop tool that reminds you to drink water, stand up, and rest your eyes during work hours. Combines interactive Windows toast notifications (with action buttons) and an optional terminal TUI dashboard.

## Goals

1. Distribute water intake across work hours with confirmation tracking
2. Remind to stand up (sedentary) and rest eyes (20-20-20 rule)
3. Only active during configurable work hours (default 9am–17pm)
4. Detect idle time and auto-pause/reset timers
5. Merge nearby notifications to avoid spam
6. Show end-of-day summary of remaining intake

## Non-Goals

- Lock screen / forced breaks
- Historical data / trends / streaks
- Cross-platform support (Windows-only for now)
- Auto-updater

## Stack

| Layer | Technology | Purpose |
|-------|-----------|---------|
| Notifications | `windows-toasts` | Interactive toast with action buttons |
| TUI | `textual` | Optional dashboard showing live progress |
| Scheduling | `apscheduler` | Timer management for all reminders |
| Idle detection | `ctypes` (Win32 `GetLastInputInfo`) | Pause when AFK |
| State | JSON file (`~/.hoa/state.json`) | Today's data |
| Config | JSON file (`~/.hoa/config.json`) | Persists across days |
| Language | Python 3.12+ | Everything |

## Architecture

```
┌──────────────────────────────────────────────┐
│              Scheduler (APScheduler)          │
│   ┌───────┐   ┌────────────┐   ┌──────────┐ │
│   │ Water │   │ Sedentary  │   │ Eye Rest │ │
│   │ Timer │   │ Timer      │   │ Timer    │ │
│   └───┬───┘   └─────┬──────┘   └────┬─────┘ │
│       └──────────────┼───────────────┘       │
│                      ▼                        │
│          Notification Engine                  │
│      (merge nearby, send toast)               │
├──────────────────────┼────────────────────────┤
│                      ▼                        │
│   Windows Toast (action buttons)              │
│   Idle Detector (GetLastInputInfo)            │
│   State (JSON file)                           │
├───────────────────────────────────────────────┤
│   TUI Dashboard (Textual) — optional          │
└───────────────────────────────────────────────┘
```

## Run Modes

1. **`hoa run`** — Headless background daemon
2. **`hoa tui`** — TUI dashboard (reads shared state file)
3. **`hoa status`** — One-shot progress print
4. **`hoa config`** — Show current config

## Timers

| Timer | Default Interval | Action Buttons |
|-------|-----------------|----------------|
| Water | 30 min | `150ml` / `250ml` / `500ml` / `Snooze` |
| Sedentary | 45 min | `Done` / `Snooze` |
| Eye Rest | 20 min | `Done` / `Snooze` |

## Idle Detection

- Poll every 30s via `GetLastInputInfo`
- Idle > 5min: pause all timers
- On resume: reset countdowns, send "Welcome back" toast

## Task Merging

- If timers due within 2min of each other, combine into single toast

## Snooze

- Default +5min, max 3 per cycle
- After max snoozes and dismiss: timer resets to full interval

## Work Hours

- Only active 09:00–17:00 (configurable)
- At 17:00: end-of-day summary toast
